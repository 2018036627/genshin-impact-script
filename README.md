# genshin-impact-script

好耶！是人畜无害的原神脚本！

## 功能

- 角色登场时自动施放元素战技
- 按下`F`键时快速拾取物品
- 按下鼠标中键后维持元素视野，再次按下后停止
- 更合理的冲刺/快跑
- 按下`S`键时自动起跳
- 起跳时自动打开风之翼

## 使用

首先，前往[ahk官网](https://www.autohotkey.com/)安装`ahk`。

然后，将[脚本](./source/index.ahk)下载至本地，并双击打开。

接下来，打开原神，在游戏窗体中按下`F1`键。听到“哔——”的一声即代表脚本已启用。

最后，按下`Alt + F4`关闭脚本。

## 注意

所有操作均绑定默认按键。
