# genshin-impact-script

好耶！是人畜无害的原神脚本！

## 功能

- 角色登场时施放元素战技
- 按下`F`键时快速拾取物品
- 按下鼠标中键后维持元素视野，再次按下后停止
- 更合理的冲刺/快跑
- 按下`S`键时自动起跳
- 起跳时打开风之翼
- 简易技能计时器
- 游戏窗口不在最前端时，将进程优先级调至最低

## 使用

首先，前往[ahk官网](https://www.autohotkey.com/)安装`ahk`。

然后，将[脚本](./source/index.ahk)下载至本地，并双击打开。

接下来，打开原神即可。

最后，按下`Alt + F4`关闭脚本。

## 注意

所有操作均绑定默认按键。
